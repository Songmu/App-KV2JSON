# NAME

kv2json - command line utility for building json

# SYNOPSIS

    % kv2json var=baz fruits[]=apple,orange aa[bb]=cc
    {"fruits":["apple","orange"],"var":"baz","aa":{"bb":"cc"}}

# DESCRIPTION

kv2json is command line utility for building json.
